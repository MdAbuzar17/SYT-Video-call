# SeeYouThere - Video Call App 📹

**SeeYouThere** is a real-time video conferencing web application that allows users to create and join secure video calls seamlessly. Built using React.js, HTML, CSS, and JavaScript on the frontend, and powered by the MERN (MongoDB, Express.js, React, Node.js) stack on the backend, this project demonstrates modern web development techniques for real-time communication.

## 🌐 Features

- 🔒 Secure one-to-one and group video calls
- 🎥 Live camera and microphone access with permission control
- 🔗 Unique room links to invite others
- 💬 Real-time chat during video sessions
- 🟢 Online user presence detection

## 🛠️ Technologies Used

### Frontend:
- HTML5, CSS3, JavaScript (ES6+)
- React.js
- WebRTC for peer-to-peer communication
- Socket.IO for real-time signaling

### Backend:
- Node.js & Express.js
- MongoDB for storing user and room data
- Socket.IO server-side integration

### Prerequisites:
- Node.js and npm installed
- MongoDB running locally or accessible via cloud (e.g., MongoDB Atlas)
  
Set up environment variables:
Create a .env file in the root with the following:
MONGO_URL=your_mongodb_connection_string
PORT=8080
